# Day 35: SQL Arithmetic Operators

**Arithmetic operators** are used to perform mathematical calculations on column values or numerical expressions.

### Common SQL Arithmetic Operators

| Operator |           Description                                       | Example |
| :---:    |           :---                                              | :--- |
| **`+`**  | **Addition:** Adds numeric values                           | `price + shipping_fee` |
| **`-`**  | **Subtraction:** Subtracts one value from another           | `original_price - discount` |
| **`*`**  | **Multiplication:** Multiplies numeric values               | `quantity * unit_price` |
| **`/`**  | **Division:** Divides one value by another                  | `total_sales / total_orders` |
| **`%`**  | **Modulus:** Returns the remainder of a division            | `total_items % items_per_box` |
